\begin{abstract}
  The continuing Big Data explosion has created a demand for efficient and scalable algorithms for similarity search.
  While much recent work has focused on \textit{approximate} $k$-NN search, \textit{exact} $k$-NN search is sometimes necessary.
  We present CAKES, a set of three novel, exact algorithms for $k$-NN search.
  CAKES's algorithms are generic over \textit{any} distance function, and rather than scaling with the cardinality or embedding dimension of the dataset, \textbf{they scale with the dataset's metric entropy and fractal dimension}.
  Furthermore, we show that for a number of real-world datasets, as opposed to a randomly-generated artificial dataset, these properties hold.
  We test these claims on datasets from the ANN-Benchmarks suite under commonly-used distance functions, as well as on a genomic dataset with Levenshtein distance and a radio-frequency dataset with Dynamic Time Warping distance.
  We demonstrate that CAKES exhibits near-constant running time as cardinality grows on data conforming to the manifold hypothesis, and has perfect recall on data in \textit{metric} spaces, while requiring an order of magnitude less indexing and tuning time compared to state-of-the-art approaches.
  We also demonstrate that CAKES exhibits significantly higher recall than state-of-the-art $k$-NN search algorithms when the distance function is not a metric.
  We conclude that CAKES is a highly efficient and scalable algorithm for exact $k$-NN search on Big Data.
  We provide a Rust implementation of CAKES.
\end{abstract}
